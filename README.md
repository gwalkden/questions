METHODOLOGICAL INFORMATION
--------------------------
Other than the readme and the license, this dataset contains the following eight things:

a) Figure1.pdf, which is Figure 1.

b) Figure2.pdf, which is Figure 2.

c) Figure3.pdf, which is Figure 3.

d) grambank_gw.csv, the end product of the data reshaping carried out by the R code.

e) grambank-values-qonly.csv, the input to the data reshaping carried out by the R code. This isn't my data, but comes from Grambank v1.0.3 (doi: 10.5281/zenodo.7844558).

f) languages.csv, containing Grambank metadata. This isn't my data, but comes from Grambank v1.0.3 (doi: 10.5281/zenodo.7844558).

g) questions-rcode.r, containing the R code used to reshape the data.

h) wlw-diachrony.csv, which is the data corresponding to Walkden, Li & Weisshaar's (2025) dataset (doi for paper: 10.18148/hs/2025.v9i2-10.185).
